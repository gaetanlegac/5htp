# 5-HTP Framework

Opinionated, Lightweight & Full Stack Typescript framework, designed for productivity, modularity and performance.

**/!\ This is a side-project. I cannot guarantee the maintenance.**

## Get Started

1. Install:

`npm i -g 5htp`

2. Create a project:

`5htp init`

3. Launch dev server:

`5htp dev`

4. Build for production:

`5htp build`
